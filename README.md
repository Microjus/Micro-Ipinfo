### Instalação
> **Clone o Repositório**
```bash
git clone https://github.com/Microjus/Micro-Ipinfo.git
```
### Permissões
> **Atribuir permissao**
```bash
chmod +x microipinfo.sh
```
### Execução
> **Execute**
```bash
bash microipinfo.sh
```
