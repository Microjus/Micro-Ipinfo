![Banner]

----
### Instalação
> **Clone o Repositório**
```bash
git clone https://github.com/Microjus/Micro-Ipinfo.git
```
### Execução
> **Atribuir permissao**
```bash
$ chmod +x microipinfo.sh
```
> **Executar**
$ bash microipinfo.sh ou ./microipinfo.sh
